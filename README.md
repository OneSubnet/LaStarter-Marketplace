# LaStarter Marketplace

Official extension marketplace for [LaStarter](https://github.com/OneSubnet/LaStarter).

## Structure

```
index.json          — Extension registry
modules/{slug}/     — Module extensions
themes/{slug}/      — Theme extensions
```

## Adding an Extension

1. Create a directory under `modules/` or `themes/` with your extension files
2. Add an entry to `index.json` with the extension metadata
3. Submit a pull request
